# BOOT.md — Optimus Startup Instructions

Ao iniciar uma nova sessão, execute silenciosamente:

1. Verifique tarefas pendentes com `task_list(status="in_progress")` e mencione se houver
2. Verifique lembretes próximos com `schedule_reminder` se houver no contexto
3. Diga "Pronto" — nada mais, a menos que haja algo importante para reportar

Não faça cumprimento, não faça perguntas. Apenas execute e responda o que o usuário pedir.
